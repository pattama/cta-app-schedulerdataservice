#### Setup a schedule
**Request**
```ruby
POST /schedules
{
  "schedule": "* * * * *", // cron string or timestamp(number)
  "rest": {
    "method": "POST",
    "url": "http://www.google.com",
    "headers": {
      "Content-Type": "application/json"
    },
    "body": {
      "nothing in real":"just to show people can add headers and body"
    }
  }
}
```

**Response**
```ruby
{
  "id": "58105c481812e42e3c844e85",
  "schedule": "* * * * *",
  "rest": {
    "method": "POST",
    "url": "http://www.google.com",
    "headers": {
      "Content-Type": "application/json"
    },
    "body": {
      "nothing in real": "just to show people can add headers and body"
    }
  }
}
```
